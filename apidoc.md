# 菜谱信息大全 API

免费

查询菜谱信息，包括简介，材料，做法

## 接口信息

**接口地址：** `https://api.pearktrue.cn/api/cookbook/`

**请求方式：** GET

**返回格式：** JSON

## 请求参数

| 参数名 | 说明         | 必填 |
| ------ | ------------ | ---- |
| search | 待搜索的菜名 | 必填 |

## 返回结果

| 字段名      | 说明                                   | 类型   |
| ----------- | -------------------------------------- | ------ |
| code        | 状态码                                 | 整数   |
| msg         | 状态信息                               | 字符串 |
| search      | 搜索的菜名                             | 字符串 |
| data        | 返回内容（数组，每个元素包含菜谱信息） | 数组   |
| name        | 菜名                                   | 字符串 |
| image       | 菜的图片 URL                           | 字符串 |
| description | 简介                                   | 字符串 |
| materials   | 材料列表                               | 数组   |
| practice    | 做法步骤列表                           | 数组   |

## 调用示例

```bash
GET https://api.pearktrue.cn/api/cookbook/?search=酸辣粉
```
